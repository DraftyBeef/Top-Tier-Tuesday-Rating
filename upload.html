File names tell the page what each file is: S3R1_Results.json, S3R1_Laps.json, S4R20_Results.json, and so on. The season and race number are read from the name (spaces, underscores, or dashes between the parts all work), so the folder layout does not matter. Drop every race from every season. The ratings are rebuilt from the full history each time, since Glicko and the Index carry across seasons. Optionally include your config.json to keep hidden drivers, trophy counts, and nicknames.
